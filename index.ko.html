파이프라인

Karpathy의 자기 컴파일 루프,
당신의 뇌를 위해.

Andrej Karpathy와 Niklas Luhmann에서 영감. 모든 입력은 같은 파이프라인을 흐릅니다. 수동 정리는 전혀 없습니다.

01
캡처
PDF 던지기, 유튜브 붙여넣기, 생각 적기. 브라우저에 파일 드래그. 모든 포맷.
02
정리
제텔카스텐 자동 분류. 인덱스 코드·태그·위키링크 자동 삽입.
03
증류
위키로 컴파일. 개념 추출. 빈틈·모순·중복 감지.
04
표현
초안 생성. 블로그·리포트·아웃라인. Claude로 AI 강화. 바로 출판.

Claude가 당신이 아는
모든 걸 기억합니다.

시작하기

5분이면 살아있는
지식 그래프.

1
전역 설치
npm install -g stellavault
Node.js 20+ 필요. 또는 npx로 설치 없이.
2
볼트 초기화
stellavault init
대화형 설정. 옵시디언 볼트를 지정하고 전체 색인.
3
Claude에 연결
stellavault setup
Claude Code·Claude Desktop·Cursor·Windsurf·VS Code에 MCP 서버 추가 — 명령 하나로.
4
그래프 실행
stellavault graph
브라우저에서 3D 신경망 그래프. 파일을 드래그해 흡수. 모바일 지원.
