**Stellavault** — Knowledge intelligence for Obsidian

Your vault remembers everything. You don't. Stellavault fixes that.

**Features:**
- Semantic search (find by meaning, not keywords)
- Memory decay tracking (FSRS algorithm — see what you're forgetting)
- Learning paths (personalized review suggestions)
- Auto-indexing (real-time, incremental)

**100% local** — SQLite + local AI embeddings. No API calls, no cloud.

**Desktop only** (needs Node.js for local embeddings)

Install: https://github.com/Evanciel/stellavault-obsidian/releases/tag/0.1.0
