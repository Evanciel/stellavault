What makes Stellavault different

🧠

AI Understands Your Notes

Local vector embeddings turn every note into searchable meaning. Hybrid BM25 + cosine + RRF fusion finds what keyword search can't.

📉

Memory Decay Tracking

FSRS algorithm tracks what you're forgetting. Get alerts before knowledge fades. Review sessions strengthen weak memories.

🌐

3D Knowledge Graph

See your entire vault as an interactive 3D neural network. Constellation view, timeline slider, cluster filters, and hand gesture control.

🔌

17+ MCP Tools

Claude Code searches your vault, tracks decisions, finds related notes, and gives daily briefings — through the Model Context Protocol.

🌌

Federation Network

Connect your vault to others via P2P. Search across the network without sharing raw text. Your knowledge stays yours.

🔒

100% Local & Private

No cloud required. No API keys. Embeddings run locally. Federation shares only encrypted vectors, never your notes.

Install

3

Explore
