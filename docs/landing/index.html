The pipeline

Four stages.
One continuous loop.

Inspired by Karpathy's self-compiling knowledge architecture. Every input flows through the same four-stage pipeline, endlessly refining itself.

◇

Capture

Drop anything into the inbox — files, URLs, voice, screenshots. No classification required.

raw/ · fleeting · voice · clips
◈

Organize

Extract concepts, tag by content, promote to literature notes. Obsidian-compatible frontmatter.

auto-tag · concept extract
◆

Distill

Detect gaps, duplicates, contradictions. Spaced-repetition decay surfaces forgotten ideas.

FSRS decay · lint · graph
✦

Express

Draft blog posts, reports, threads, Instagram carousels — grounded in your own knowledge.

draft · compile · ask

Install, point, explore.

Install

One command. Node 20+. Works on macOS, Linux, and Windows.

$ npm i -g stellavault

Initialize

Interactive wizard points Stellavault at your Obsidian vault and builds the first index.

$ stellavault init

Explore

Launch the 3D graph in your browser. Click any node to travel, search, or drop new notes.

$ stellavault graph
