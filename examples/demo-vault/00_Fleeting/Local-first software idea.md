# Local-first Software Idea

What if all knowledge tools worked offline-first, with optional sync?

Benefits:
- No vendor lock-in (your data = your files)
- Privacy by default (nothing leaves your machine)
- Works on airplane, in tunnels, anywhere
- Fast (no network latency for queries)

Challenge: collaboration requires some network layer, but it can be opt-in and privacy-preserving (share embeddings, not raw text).

This is basically what Stellavault does. The federation protocol shares only similarity scores and 50-char snippets.
